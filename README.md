# 🛒 Hyperlocal Price Intelligence System

A real-time price tracking and prediction system for e-commerce products.

## 🚀 What This Does
- Scrapes product prices from Amazon automatically
- Stores price history in a database
- Analyzes price trends over time
- Predicts best time to buy using ML

## 🛠️ Tech Stack
- Python, BeautifulSoup, Requests
- SQLite, Pandas
- Prophet (coming in Week 3)
- Streamlit Dashboard (coming in Week 4)

## 📊 Project Status
- [x] Web Scraper
- [ ] EDA & Visualization
- [ ] ML Price Prediction
- [ ] Live Dashboard
